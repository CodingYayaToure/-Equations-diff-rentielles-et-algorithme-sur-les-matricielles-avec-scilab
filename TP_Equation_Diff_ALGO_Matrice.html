EXEMPLE 1

Pour rsoudre l'quation diffrentielle donne $ y'' = -4y $ avec les conditions initiales $ y(0) = 3 $ et $ y'(0) = 0 $, nous pouvons transformer cette quation d'ordre 2 en un systme de deux quations d'ordre 1.

tapes :

  1. Introduire de nouvelles variables :

    Posons $y_1 = y$ et $y_2 = y'$.

  2. Rcrire l'quation d'ordre 2 :

    L'quation $ y'' = -4y $ devient $ y_2' = -4y_1 $.

    Nous avons aussi $ y_1' = y_2 $.

  3. Former le systme d'quations d'ordre 1 :

    Le systme devient : $ \begin{cases} y_1' = y_2 \\ y_2' = -4y_1 \end{cases} $

  4. Conditions initiales :

    Les conditions initiales sont : $ \begin{cases} y_1(0) = 3 \\ y_2(0) = 0 \end{cases} $

Explications :

  1. Dfinir la fonction odefun :

    • dydt = [y(2); -4*y(1)] reprsente le systme d'quations diffrentielles.

  2. Conditions initiales :

    • y0 = [3; 0] reprsente les conditions initiales $ y_1(0) = 3 $ et $ y_2(0) = 0 $.

  3. Intervalle de temps :

    • t = linspace(0, 10, 100) dfinit un intervalle de temps de 0 10 avec 100 points.

  4. Rsoudre le systme :

    • y = ode(y0, t, odefun) utilise la fonction ode pour rsoudre le systme d'quations diffrentielles.

  5. Tracer les solutions :

    • plot(t, y(1, :), 'r-', t, y(2, :), 'b--') trace les solutions $ y_1 $ et $ y_2 $ sur le mme graphique.

  6. Lgendes et titres :

    • legend("$y_1 = y$", "$y_2 = y'$") ajoute des lgendes aux courbes en utilisant des notations mathmatiques.
    • xlabel, ylabel, et title ajoutent des lgendes et des titres appropris au graphique.

// Utiliser funcprot pour viter la redfinition de la fonction
funcprot(0);

// Dfinir la fonction qui reprsente le systme d'quations
function dydt = system(t, y)
    dydt = zeros(2, 1); // Initialiser le vecteur de sortie
    dydt(1) = y(2);     // y' = v
    dydt(2) = -4 * y(1); // v' = -4y
endfunction

// Conditions initiales
y0 = [3; 0]; // y(0) = 3, y'(0) = 0

// Intervalle de temps pour la solution
t0 = 0; // Temps initial
tf = 10; // Temps final
t = linspace(t0, tf, 100); // Crer un vecteur de temps

// Rsoudre le systme d'quations avec ode
y = ode(y0, t0, t, system);

// Tracer les rsultats
clf; // Effacer la figure prcdente
plot(t, y(1, :), 'r', 'LineWidth', 2); // Tracer y(t)
plot(t, y(1, :), 'r-', t, y(2, :), 'b--'); // Tracer y1 et y2 sur le mme graphique

xlabel('Temps t');
ylabel('y(t)');
title('Solution de l''quation diffrentielle y'''' = -4y'); // chapper l'apostrophe
// Lgendes et titres
legend("y_1 = y", "y_2 = y''"); // Ajouter des lgendes aux courbes
xgrid; // Activer la grille

// Dfinir la fonction odefun pour le systme de Van der Pol
function dydt = odefun(t, y)
    c = 0.1; // Paramtre c
    dydt = [y(2); c * (1 - y(1)^2) * y(2) - y(1)]; // Systme d'quations diffrentielles
endfunction

// Conditions initiales
y0 = [2; 0]; // y(0) = 2 et y'(0) = 0

// Intervalle de temps
t = linspace(0, 20, 1000); // Crer un vecteur de temps de 0  20 avec 1000 points

// Rsoudre le systme
y = ode(y0, 0, t, odefun); // Utiliser la fonction ode pour rsoudre le systme d'quations

// Tracer les solutions
clf; // Effacer la figure prcdente
plot(t, y(1, :), 'r-', t, y(2, :), 'b--', 'LineWidth', 4); // Tracer y1 et y2 sur le mme graphique

// Lgendes et titres
legend('u_1 = y(t)', 'u_2 = y''(t)'); // Ajouter des lgendes aux courbes
xlabel('Temps (t)'); // tiqueter l'axe des x
ylabel('Valeur'); // tiqueter l'axe des y
title('Solutions de l''quation de Van der Pol'); // Titre du graphique
xgrid; // Activer la grille
